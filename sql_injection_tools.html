SQL Injection Tools

      sqlmap -> open source pentest tool for detecting and exploiting SQL injection flaws
      Mole -> automatic SQL injection exploitation tools; only by providing a vulnerable URL and a valid string on the site, it can detect the injection and exploit it using the UNION technique of a Boolean query-based technique
      Blisqy -> exploits time-based blind SQL injection in HTTP headers (MySQL/MariaDB); attackers use Blisqy to find a potential time-based blind SQL injection and then prepare a script to exploit the vulnerable web application
