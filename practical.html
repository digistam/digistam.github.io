TRYHACKME rooms

In order to work with TryHackMe it's very convenient to create a VPN connection between Kali machine and TryHackMe.

      sudo apt install openvpn
      Download the vpn configuration file from TryHackMe.com access page
      locate the full path to the downloaded VPN configuration file (most of the times downloaded to the Download folder)
      sudo openvpn /path-to-file/file-name.ovpn
      Deploy a machine in TryHackMe, take a note of the IP address and ping the IP address / browse to http://IP-address
