Social Engineering

Social Engineering Concepts

Social engineering is the art of manipulating people to divulge sensitive information to use it to perform some malicious action. It targets the weakness of people. Attackers take a special interest in developing social engineering skills and can be so proficient that the victims might not even notice the fraud. Attackers ensure they know the organization's perimeter and the people on its perimeter such as security guards, receptionists, help-desk workers etc.

Human-based Social Engineering

      Impersonation -> attacker pretends to be legitimate or authorized person
      Vishing -> (voice or VoIP phishing) impersonation using voice to trick individuals, using Caller ID spoofing, pre-recorded messages and instructions resembling a legitimate financial institution
      Eavesdropping -> listening to a conversation or reading others' messages
      Shoulder surfing -> looking over someone's shoulder as the key information into a device
      Dumpster diving -> retrieving sensitive information by searching through trash bins
      Reverse Social Engineering -> perpetrator assumes the role of a knowledgeable professional so that the organization's employees ask them for information, the attacker manipulates questions to draw out the required information
      Piggybacking -> entry into a building or area with consent of the authorized person (hey, I forgot my ID badge, please let me in!)
      Tailgating -> entry into a building or area without consent of the authorized persion (wearing a fake badge, closey following an authorized person through a door that requires key access)
      Diversion Theft -> delivery professionals or transport companies deliver consignement to a wrong location where the goods are stolen
      Honey Trap -> attractive person begins a fake relationship with the victim
      Baiting -> attackers offer something alluring in exchange for information (also: usb flash drive dropped on the floor)
      Quid Pro Quo -> something for something, attackers keep calling random numbers claiming to be calling from technical support, baiting technique
      Elicitation -> extracting specific information from the victim by involving him in normal and disarming conversations, using good social skills

Computer-based Social Engineering

      Phishing -> the attacker registers a fake domain name, builds a lookalike website and mails the website link to users
      Spear Phising -> specialized social engineering content directed at specific employee(s)
      Whaling -> phishing high profile executives like CEO, CFO, politicians, celebrities
      Pharming -> redirecting urls to attacker controlled websites using DNS cache poisoning and host file modification, phishing without a lure
      Spimming -> spam over instant messaging
      Angler Phishing -> targetting disgruntled users or customers over sociale media platforms
      Catfishing -> identity theft, creating a fake profile and masquerade as the owner of the account, asking for money
      Deepfake attack -> phishing using deepfake, blackmail etc.
      Spam mail
      Instant Chat messenger
      Pop-up window attacks -> clicking a hyperlink button that redirects to fake web pages asking for personal information
      Hoax letters
      Chain letters -> pyarmid games
      Scareware

Mobile-based Social Engineering

      Publishing malicious apps -> creating a malicious app and publishing it on major application stores using popular names
      Repackaging legitimate apps -> downloading legitimate app, repack it with malware and uploading it to third party application store
      Using fake security applications -> pop-ups telling the end user they need to download an application to their phone
      SMiShing (SMS Phishing) -> luring users to malicious website

Impersonation on Social Networking Sites

Social Engineering through impersonation on social networking sites

Two ways impersonation occurs:

      Creating fictitious profile of the victim
      Stealing victim's password or indirectly gaining access to the victim's social media account

Impersonation on Facebook

      Create a fake user group on Facebook identified as "Employees of" the target company
      Using a false identity, proceed to "friend" or invite actual employees
      Users join the group and provide their credentials such as date of birth, educational and employment backgrounds etc.
      Using the details of any one of the employees, an attacker can compromise a secured facility to gain access to the building

The Social-Engineer Toolkit (SET) is an open source Python driven tool aimed at penetration testing via social engineering. It is freely available and can be used to carry out a range of attacks. It allows attackers to draft email messages, attach malicious files and send them to a large number of people using spear phising. SET's multi-attack method allows Java applets, the Metasploit browser, Credential Harvester/Tabnabbing to be used simultaneously.

. It serves as a comprehensive toolbox for professionals to understand, simulate, and test the vulnerabilities associated with social engineering. SET offers an extensive range of tools and functionalities tailored to craft and deploy diverse social engineering attacks. These attacks encompass phishing campaigns, credential harvesting, the creation of malicious websites, and the generation of exploitative payloads, all engineered to exploit human psychology rather than solely relying on technical vulnerabilities.
