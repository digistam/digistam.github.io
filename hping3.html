Hping3 commands

Hping3 cheat sheet

https://cyberwar.nl/d/cheatsheets/hping3_cheatsheet_v1.0-ENG.pdf

Most useful:

      -c --count -> count response packets
      -i --interval -> secs
      -d --data -> datasize packet body size
      -E --file -> insert file into packet's data (target machine must be listening !)
      -T --traceroute
      -a --spoof -> spoof IP address (manually)
      --rand-source -> spoof IP address (randomly)
      -t --ttl -> set ttl value
      -w --win -> set windows size (64)
      -f --frag -> split packets (16 bytes)

TCP Flags:

      -S --syn
      -A --ack
      -F --fin
      -R --rst
      -P --push
      -U --urg
      -X --xmas

Protocol selection:

      -1 --icmp
      -2 --ucp
      -0 --rawip
      -8 --scan
      -9 --listen

Wireshark

Using wireshark while using hping3 will give a good insight in what's going on!

Enumeration

Scan open ports on target

      sudo hping3 -8 (--scan) 0-1000 -S 192.168.1.254 -i 3 | grep -v 'Not'
