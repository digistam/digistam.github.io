Challenge: Web Warriors Defend Against the OWASP Menace

Scenario

You are a security analyst hired by a reputed art gallery, Impression House, which specializes in collecting art materials and providing an e-commerce platform for renowned artists. Recently, they have suffered multiple security breaches due to various web application vulnerabilities, impacting their customer data and financial transactions. The company suspects attackers have exploited several web applications and configuration vulnerabilities. Your task is to assess their web infrastructure and uncover these vulnerabilities before malicious actors can further exploit them. You will be investigating OWASP's Top 10 Web Application vulnerabilities. You must assess the organization's web application for attacks, including Arbitrary File Upload, CMS vulnerabilities, Password Cracking, System Misconfiguration, and Privilege Escalation. The challenge will simulate real-world attacks, and you'll need to identify, exploit, and mitigate these vulnerabilities to strengthen Impression House's defenses.

Description

This Challenge requires you to exhibit knowledge in web applications, red team methodologies, arbitrary file upload and Exploitation, and WordPress Plugin exploitation using various tools. Demonstrate network scanning skills, password cracking, and credential attacks. The target machine runs on a Linux Ubuntu OS with an IP address of 10.10.1.69. To get the flag(s), you must obtain administrative privilege on the machine by performing various attack techniques.
