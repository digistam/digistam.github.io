Research

Vulnerability research is the process of analyzing protocols, services and configurations to discover vulnerabilities and design flaws that will expose an operating system and its applications to exploit, attack or misuse.

      Gather information about security trends
      Notice newly discovered threats
      Attack surfaces, Attack vectors, Techniques
      Find weaknesses in the OS and applications
      Know how to recover from an attack

Exploiting

      Identify the vulnerability
      Determine the risk associated with the vulnerability
      Determine the capability of the vulnerability
      Develop the exploit
      Select the method for delivering -> local or remote
      Generate and deliver the exploit
      Gain remote access

    Vulnerability Management Life Cycle

    Important process that helps identify and remediate security weaknesses before they can be exploited.

    Pre-Assessment Phase: prepatory phase, defining policies and standards, clarifying scope of the assessment and creating a good baseline for vulnerability management. Define approved ports, software, drivers, basic configuration of each system. Understand the network archicture and infrastructure and identify the controls already in place.
