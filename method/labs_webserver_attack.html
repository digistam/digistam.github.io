Web Server Attack

After gathering required information about the target web server, the next task is to attack the web server.

Crack FTP credentials using a Dictionary Attack

      Find open FTP port with nmap
      sudo nmap -p21 domain.tld
      If port 21 is open, check if an FTP server is hosted on the machine
      ftp domain.tld
      The need for credentials implies that an FTP server is hosted on the machine
      Guess some usernames and passwords
      Next, attempt to gain access to the FTP server performing a dictionary attack with THC Hydra
      sudo hydra -L /home/attacker/Desktop/Wordlists/Usernames.txt -P /home/attacker/Desktop/Wordlists/Passwords.txt ftp://domain.tld
      Hydra tries various combinations of usernames and passwords present in the Wordlists
      On completion of the password cracking, the cracked credentials appear; try to log in to the FTP server using one of the cracked username and password combinations
