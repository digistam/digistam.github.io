Simple Bluetooth Discovery

Scan BLE devices

      service bluetooth start
      sudo bluetoothctl
      scan le
      scan stop

Create pcap file

      console -> Actions -> Split Terminal Vertically
      sudo btmon -w /home/username/Downloads/demo_ble_cap.pcap
      In right terminal screen: bluetoothctl -> scan le
      After a while: exit
      In left terminal screen: btmon -> CTRL + C
