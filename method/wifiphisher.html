WiFiPhisher

Wifiphisher is simple to install and works out of the box!

Installation

      sudo su
      apt-get install -y libnl-3-dev libnl-genl-3-dev libssl-dev python3-virtualenv
      cd /opt/
      git clone https://github.com/wifiphisher/wifiphisher.git
      cd wifiphisher
      virtualenv -p python3 env
      source env/bin/activate
      python3 -m pip install ConfigParser
      git clone https://github.com/wifiphisher/roguehostapd.git
      cd /opt/wifiphisher/roguehostapd
      python3 setup.py install
      cd /opt/wifiphisher
      python3 -m pip install six
      python3 -m pip install .

Start WiFi AP

wifiphisher -eN

p>

with the command above you can select an AP in the neighbourhood and spoof it with multiple options

wifiphisher -e NogEenTest -p wifi_connect -kB -nE

with the command above you can create a free AP to lure victims
