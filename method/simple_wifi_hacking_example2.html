Simple WiFi hacking with Aircrack-ng

First, check if everything is well with airmon-ng

sudo airmon-ng check

In case of any conflicts -> sudo airmon-ng check kill

Now, start monitoring mode -> sudo airmon-ng start wlan0

Show detectable APs with airodump-ng

sudo airodump-ng wlan0

You can select a specific BSSID from the list:

sudo airodump-ng wlan0 --bssid [BSSID]

You can select a specific channel to monitor:

sudo airodump-ng wlan0 --channel 6

It's possible to write to a .cap file:

sudo airodump-ng wlan0 --bssid [BSSID] -w output

The .cap file can be analyzed with Wireshark
