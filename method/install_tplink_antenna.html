How to install TP-Link AC600 Drivers on Kali Linux (Archer T2U Plus)

Youtube: Instruction

      sudo apt update && sudo apt upgrade
      sudo apt-get install dkms
      sudo apt-get install bc
      sudo apt-get install rsync
      sudo apt-get install git
      sudo apt update
      sudo reboot now

After installing the packages, put the TP-Link antenna in the USB port. Assign the usb device to the VMware client and not to the host. Type lsusb in the terminal. The antenna has to show up in the output.
