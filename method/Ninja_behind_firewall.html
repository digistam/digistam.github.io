Ninja behind NAT, firewall and proxy

Jonin

      Download Ninja Jonin
      Jonin is a fully customizable CLI application to communicate with and send commands to Ninja running behind any NAT, firewall and proxy! Providing secure shell access, file transfer and shell stream (stream shell output from remote to a local file)
      Jonin has no prerequisites, you can just download the release and use it right away!
      Jonin is the controller (commander), you would need Ninja on target (remote) computer(s) to host and execute the Jonin's commands

Download Jonin

https://github.com/ErAz7/Jonin/releases

Dowload Ninja

https://github.com/ErAz7/Ninja/releases

Create Ninja package

      Extract Ninja-v1.2.1-win.zip
      open config folder and right-click on constants.json
      Open with Notepad++
      change PORT in config/constants.json, ports should be the same
      forward configured port on Jonin's router
      change HOST to the IP address of the Jonin computer (attacker)

Start Jonin listener

Double click Jonin-v1.1.0-win.exe to start the listener

Send Ninja to target/remote computer

Run Ninja on the target/remote computer, run Jonin on controller/local computer and wait for some report from your Ninja !

Commands

      list -> Enter
      connect 1
      change -> Enter -> Type: cmd
      ipconfig
      whoami
