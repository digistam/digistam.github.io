Upload files to Github

Procedure

It can be important to upload tools to Github, especially where it comes to hacking tools (there are a lot of infected files out in the open) and the tools can't be downloaded from Github itself. When you have a good, thorougly tested tool, make a backup of it in a personal Github repo. Unfortunately, you can't upload files directly using the webbrowser.

      Download Github installer
      Copy the files to a directory on the computer
      cd to this directory
      git init
      git status
      git add
