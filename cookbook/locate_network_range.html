Locate Network range

If you have a domain name of an organization and you want to find out the network range:

Method 1: dnsrecon and RIPE website

      dnsrecon -d belastingdienst.nl
      Copy IP address of "A" record
      Visit RIPE Database
      Paste the IP address and search
      Note the value at inetnum

Method 2: Censys website

      Log in @ censys.io
      Search for the domain
      Click the right match
      Click through on Routing
