Ransomware

Ransomware is a type of malware that restricts access to the infected computer system or critical files and documents stored on it, and then demands an online ransom payment to the malware creator(s) to remove user restrictions. Ransonware might encrypt files stored on the system's hard disk or lock the system and display messages meant to trick the user into paying the ransom.

Usually, ransomware spreads as a Trojan, entering a system through email attachments, hacked websites, infected programs, app downloads from untrusted sites, vulnerabilities in network servers etc. After infection, the payload in the ransomware runs and encrypts the data (files and documents) which can be decrypted only by the malware author.

In a web browser, a text file or webpage displays the ransomware demands. The victim's fear, trust, surprise and embarrassment is the reason the ransom is often payed.

Example: BlackCat

      Written in Rust
      Profoundly known as ALPHA, AlphaVM, AlphaV
      Targets almost all OSes (Windows, Linux, ESXI VM)
      Supplied as Ransomware as a service (RaaS)
      Using Blackcat, attackers can target various IT industries worldwide
      Blackcat employs phishing tactics and delivers payload using vulnerable applications and exposed toolsets

Example: BlackMatter

      Written in C
      Extension of DarkSide and REvil
      Targets mainly Windows but can also compromise Linux
      Supplied as Ransomware as a service (RaaS)
      BlackMatter crashes various files and closes processes
      Attackers can also gain control over domain controllers, ACLs and User Access Controls (UAC)
